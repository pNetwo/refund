# Refund

É uma aplicação web de pedidos de reembolso e listagem de despesas empresariais.

![Preview do Projeto](/web/src/assets/screens/signin.png)
![Preview do Projeto](/web/src/assets/screens/employee.png)![Preview do Projeto](/web/src/assets/screens/manager.png)


## 📋 Descrição

Criei esse projeto prático na trilha Full-Stack da Rocketseat. 
É uma aplicação completa que envolve a conexão entre front-end em React e a API em Node. Esse projeto aborda a criação de contas, validações e rotas de autenticação. Também foi explorada a paginação e pesquisa de solicitações.

## 🛠️ Tecnologias Utilizadas

### **Front-End**
- **React**
- **TypeScript**
- **Tailwind**

### **Back-end**
- **Express**
- **Node.js**
- **Prisma**
- **Zod**
